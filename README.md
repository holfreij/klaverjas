# Klaverjas Online

Online multiplayer Klaverjas card game (Rotterdam rules) as a Progressive Web App.

## Features

- Play Klaverjas with 4 players online
- Spectator mode to watch games
- "Table device" mode for local play with phones + shared screen
- Manual roem (bonus) claiming for authentic gameplay
- No accounts required - just join and play

## Development

```bash
npm install
npm run dev
```

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run test` | Run unit tests |
| `npm run check` | TypeScript type checking |

## Tech Stack

- **Framework**: SvelteKit
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Real-time**: Firebase Realtime Database
- **Testing**: Vitest
- **Deployment**: GitHub Pages
